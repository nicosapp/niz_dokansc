Create quickly and easily a beautiful dynamic Dokan stores carousel for your marketplace website.

== Description ==

With Niz Stores Carousel for Dokan include stores carousel in your pages.

With a simple shortcode create wonderful stores carousel for your marketplace!

Customize your carousel with the following attributes:
* Featured stores or not
* Number of products in the carousel
*Number of products visible at once in the carousel
* Autoplay
* Pause
* Loop
* Navigations buttons
* Dots buttons
* Speed of the carousel
* Delay between swipes

With one shortcode for each carousel, create as many carousels as you want in the same page and/or different pages!

== Frequently Asked Questions ==

= Does the carousel use in-built dokan store template? =

Yes, carousel are built from the dokan store loop tempate. If you customize this template, changes will appear in the carousel.

= Can I create multiple stores carousels on the same page? =

Yes, you can create several carousels. Thanks to the Niz shortcode generator, create your custom shortcode and copy-paste it wherever you want on your pages.
